# Fluentia — Project Overview

## Product Summary
Fluentia is a premium, AI-powered English speaking coach for students who can understand English but lack confidence in speaking. The app helps users practice real-world conversation, improve fluency, and learn how to speak appropriately in different contexts.

## Product Vision
Create a confidence-building language learning experience that feels modern, immersive, and intelligent.

## Core Differentiators
- Context-aware speaking practice for formal and casual situations
- Tone analysis for polite, direct, and appropriate communication
- Confidence detection across user responses
- Scenario-based practice for real-life situations
- Positive, encouraging AI feedback
- Safe learning with content moderation and misuse handling
- Premium dark 3D interface with depth and motion

## Key Features
- Initial speaking assessment
- Beginner / Intermediate / Advanced level assignment
- Scenario selection and mode selection
- Text and simulated voice input
- Grammar, vocabulary, fluency, pronunciation, tone, and confidence feedback
- Fluency score and confidence meter
- Simple and advanced improved responses
- Safety moderation for inappropriate input
- Real AI mode and simulated prototype mode

## Success Criteria
- Users understand their level within the first session
- Users complete at least one speaking scenario without confusion
- Feedback feels supportive and realistic
- The UI appears premium, modern, and easy to navigate
- Safety rules reliably block inappropriate language

## Product Principles
1. Clarity before complexity
2. Confidence before correction
3. Premium visuals without clutter
4. Real-world learning over generic chatbot behavior
5. Safety and age-appropriate interactions first
